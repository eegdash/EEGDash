**Install**

```bash

```

**Access the data**

```python
from eegdash.dataset import DS006159

```

### Query & Filter

**Filter by subject**

```python
dataset = DS006159(cache_dir="./data", subject="01")
```

**Advanced query**

```python
dataset = DS006159(
    cache_dir="./data",

)
```

**Iterate recordings**

```python
for rec in dataset:

```
