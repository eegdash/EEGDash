**Access the data**

```python
from eegdash.dataset import NM000234

dataset = NM000234(cache_dir="./data")

```

### Query & Filter

**Filter by subject**

```python
dataset = NM000234(cache_dir="./data", subject="01")
```

**Advanced query**

```python
dataset = NM000234(
    cache_dir="./data",
```

**Iterate recordings**

```python
for rec in dataset:
```
