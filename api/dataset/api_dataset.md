# Datasets API

The `eegdash.dataset` package exposes dynamically registered dataset
classes. See [eegdash.dataset package](eegdash.dataset.md) for the module-level API, including
[`EEGChallengeDataset`](eegdash.dataset.EEGChallengeDataset.md#eegdash.dataset.EEGChallengeDataset) and helper utilities.

## Dataset Overview

EEGDash currently exposes **1106 OpenNeuro EEG datasets** that are
registered dynamically from mongo database. The table below summarises
the distribution by experimental type as tracked in the summary file.
