### Get Started

**Install**

**Access the data**

```python
from eegdash.dataset import NM000210

dataset = NM000210(cache_dir="./data")
```

### Query & Filter

**Filter by subject**

```python
dataset = NM000210(cache_dir="./data", subject="01")
```

**Advanced query**

```python
dataset = NM000210(
    cache_dir="./data",
    query={"subject": {"$in": ["01", "02"]}},
)
```

**Iterate recordings**

```python
for rec in dataset:
```
