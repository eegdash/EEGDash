### Get Started

**Install**

```bash

```

**Access the data**

```python
from eegdash.dataset import DS002722
```

### Query & Filter

**Filter by subject**

```python
dataset = DS002722(cache_dir="./data", subject="01")
```

**Advanced query**

```python
dataset = DS002722(
```
