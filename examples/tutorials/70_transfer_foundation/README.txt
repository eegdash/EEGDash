Transfer, Foundation Models, and EEG2025
=========================================

Four advanced tutorials and project examples for transfer learning,
foundation-model fine-tuning, and the EEG2025 Foundation Challenge.
Sourced from ``docs/tutorial_restructure_plan.md`` Category H
(lines 458-470).

Run them in order:

1. ``plot_70_challenge_dataset_basics.py`` — use ``EEGChallengeDataset``
   and understand how it differs from ``EEGDashDataset``.
2. ``plot_71_cross_task_transfer.py`` — EEG2025 Challenge 1 (cross-task
   transfer from resting-state to contrastChangeDetection).
3. ``plot_72_subject_invariant_regression.py`` — EEG2025 Challenge 2
   (predict p-factor with subject-invariant regression).
4. ``plot_73_finetune_pretrained_model.py`` — fine-tune a Braindecode
   pretrained model when a stable example is available.
