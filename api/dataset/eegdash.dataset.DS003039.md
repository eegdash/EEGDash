**Access the data**


### Query & Filter

**Filter by subject**

```python
dataset = DS003039(cache_dir="./data", subject="01")
```

**Advanced query**

```python
dataset = DS003039(

    query={"subject": {"$in": ["01", "02"]}},
)
```

**Iterate recordings**

```python
for rec in dataset:
```

## About This Dataset

This free walking experiment consists of 19 participants.
Subjects walked two different routes outdoors
Each route was walked twice. Once while pressing buttons in a self-paced manner.
The article (see Reference) contains all methodological details
- Nadine Jacobsen (March, 2020)
