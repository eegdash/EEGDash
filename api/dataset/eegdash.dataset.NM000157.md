### Get Started

**Install**

```bash
pip install eegdash
```

**Access the data**

```python
from eegdash.dataset import NM000157

dataset = NM000157(cache_dir="./data")
# Get the raw object of the first recording

```

### Query & Filter

**Filter by subject**

```python
dataset = NM000157(cache_dir="./data", subject="01")
```

**Advanced query**

```python
dataset = NM000157(
    cache_dir="./data",
    query={"subject": {"$in": ["01", "02"]}},
)
```

**Iterate recordings**

```python
for rec in dataset:

```
