### Get Started

**Install**

```bash

```

**Access the data**

```python
from eegdash.dataset import HBN_r7_bdf

dataset = HBN_r7_bdf(cache_dir="./data")
# Get the raw object of the first recording
```

### Query & Filter

**Filter by subject**


**Advanced query**
