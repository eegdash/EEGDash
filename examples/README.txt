EEGDash Examples & Tutorials
===========================

This directory contains the source code for the EEGDash examples gallery.
Each Python script is a standalone tutorial that executes during the
documentation build.

Categories:
-----------

- **tutorials/**: The curated learning path, from install to evaluation.
- **how_to/**: Task-focused recipes for specific operational questions.
- **applied/**: Real-world research case studies (age, sex, p-factor).
- **eeg2025/**: Official Foundation Challenge starter kits.
- **hpc/**: Scaling EEGDash on shared clusters and SLURM.
- **core/**: Minimal, linear scripts for rapid prototyping.

For the rendered version with figures, see: https://eegdash.org/generated/auto_examples/index.html
