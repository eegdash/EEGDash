Start Here
==========

Three short tutorials that take you from "I have eegdash installed" to a
working PyTorch ``DataLoader`` over real EEG records. CPU-only, designed to run
in under a few minutes each. Sourced from
``docs/tutorial_restructure_plan.md`` Category A.

Run them in order:

1. ``plot_00_first_search.py`` — find datasets and records.
2. ``plot_01_first_recording.py`` — load one recording and inspect it.
3. ``plot_02_dataset_to_dataloader.py`` — turn an ``EEGDashDataset`` into
   windows and a PyTorch ``DataLoader``.
