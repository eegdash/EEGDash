### Get Started

**Install**

```bash

```

**Access the data**

```python
from eegdash.dataset import DS004012

dataset = DS004012(cache_dir="./data")

```

### Query & Filter

**Filter by subject**

```python
dataset = DS004012(cache_dir="./data", subject="01")
```

**Advanced query**

```python
dataset = DS004012(
    cache_dir="./data",
    query={"subject": {"$in": ["01", "02"]}},
)
```

**Iterate recordings**

```python
for rec in dataset:
```
