MUSING: EEG dataset#

Dataset from OpenNeuro.

Access recordings and metadata through EEGDash.

Citation: Unknown (—). **.

Modality: — Subjects: — Recordings: — License: — Source: OpenNeuro

Metadata: Limited (0%)

Quickstart#

Install

pip install eegdash

Access the data

from eegdash.dataset import MUSING

dataset = MUSING(cache_dir="./data")
# Get the raw object of the first recording
raw = dataset.datasets[0].raw
print(raw.info)

Filter by subject

dataset = MUSING(cache_dir="./data", subject="01")

Advanced query

dataset = MUSING(
    cache_dir="./data",
    query={"subject": {"$in": ["01", "02"]}},
)

Iterate recordings

for rec in dataset:
    print(rec.subject, rec.raw.info['sfreq'])
