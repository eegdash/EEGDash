# Datasets API

The `eegdash.dataset` package exposes dataset classes that are
registered dynamically at import time. See [eegdash.dataset package](eegdash.dataset.md) for the
module-level API, including [`EEGChallengeDataset`](eegdash.dataset.EEGChallengeDataset.md#eegdash.dataset.EEGChallengeDataset)
and helper utilities.

## What’s in the registry

EEGDash exposes **700+ OpenNeuro EEG datasets**, registered dynamically
from MongoDB. The table below summarizes the breakdown by experimental
type (847 datasets in this build).
