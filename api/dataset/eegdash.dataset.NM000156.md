### Get Started

**Install**

```bash
pip install eegdash
```

**Access the data**

```python
from eegdash.dataset import NM000156

dataset = NM000156(cache_dir="./data")

```

### Query & Filter

**Filter by subject**

```python
dataset = NM000156(cache_dir="./data", subject="01")
```

**Advanced query**

```python
dataset = NM000156(
    cache_dir="./data",
    query={"subject": {"$in": ["01", "02"]}},
)
```

**Iterate recordings**

```python
for rec in dataset:
```
