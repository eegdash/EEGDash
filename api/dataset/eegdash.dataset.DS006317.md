## About This Dataset

This dataset is a imagined speech dataset with two participants, identified as sub-01 to sub-02. The dataset includes raw data in edf formats. Information also can be found in [https://github.com/baoyudu/How-Far-to-NISN](https://github.com/baoyudu/How-Far-to-NISN) .
