**Install**

**Access the data**

```python
from eegdash.dataset import DS003703
```

### Query & Filter

**Filter by subject**

```python
dataset = DS003703(cache_dir="./data", subject="01")
```

**Advanced query**

```python
dataset = DS003703(
    cache_dir="./data",
    query={"subject": {"$in": ["01", "02"]}},
)
```

**Iterate recordings**

```python
for rec in dataset:

```
