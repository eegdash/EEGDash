Event-Related Decoding
======================

Two tutorials covering tasks where labels come from events and annotations
(visual and auditory oddball paradigms). Sourced from
``docs/tutorial_restructure_plan.md`` Category C.

Run them in order:

1. ``plot_20_visual_p300_oddball.py`` — P3 target vs standard classifier.
2. ``plot_21_auditory_oddball.py`` — auditory oddball framed as a contrast
   with the visual P300 case.
