### Get Started

**Install**

```bash

```

**Access the data**


### Query & Filter

**Filter by subject**

```python
dataset = DS002791(cache_dir="./data", subject="01")
```

**Advanced query**

```python
dataset = DS002791(
```

**Iterate recordings**

