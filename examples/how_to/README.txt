How-To Guides
=============

Task-focused recipes for specific EEGDash workflows. Unlike the
tutorials in ``examples/tutorials``, these guides assume you already
know the basics and want a direct answer to a single question
(for example, "how do I work offline?"). Sourced from
``docs/tutorial_restructure_plan.md`` Category I (lines 615-621,
1021-1037).

Cross-link with the HPC track (``examples/hpc``) where appropriate.
