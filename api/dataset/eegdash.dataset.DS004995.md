### Get Started

**Install**

```bash

```

**Access the data**

```python
from eegdash.dataset import DS004995

dataset = DS004995(cache_dir="./data")
```

### Query & Filter

**Filter by subject**

```python
dataset = DS004995(cache_dir="./data", subject="01")
```

**Advanced query**

```python
dataset = DS004995(
    cache_dir="./data",
```

**Iterate recordings**

