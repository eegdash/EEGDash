Applied Research Projects
=========================

These are realistic project-style examples adapted from
``docs/tutorial_restructure_plan.md`` Category G ("Applied Examples
To Keep But Reframe", lines 1052-1100). They are *not* first-week
tutorials. Each script assumes you have already worked through the
"Start Here" trio (``examples/tutorials/00_start_here``) and the
core workflow path (``examples/tutorials/10_core_workflow``).

Compared with tutorials, these projects:

- target a concrete scientific question (age, p-factor, sex, P300
  transfer, clinical catalog summary);
- use realistic data sizes and runtimes;
- emphasize labels, splits, baselines, and limitations rather than
  individual EEGDash API calls.

Treat them as starting points for your own analyses, not as
prescriptive recipes.
