### Get Started

**Install**

```bash
pip install eegdash
```

**Access the data**

```python
from eegdash.dataset import DS004660

dataset = DS004660(cache_dir="./data")
# Get the raw object of the first recording

```

### Query & Filter

**Filter by subject**

```python
dataset = DS004660(cache_dir="./data", subject="01")
```

**Advanced query**

```python
dataset = DS004660(
    cache_dir="./data",
    query={"subject": {"$in": ["01", "02"]}},
)
```

**Iterate recordings**

```python
for rec in dataset:

```
