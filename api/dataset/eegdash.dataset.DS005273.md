### Get Started

**Install**

```bash

```

**Access the data**

```python
from eegdash.dataset import DS005273
```

### Query & Filter

**Filter by subject**

```python
dataset = DS005273(cache_dir="./data", subject="01")
```

**Advanced query**

```python
dataset = DS005273(
```
