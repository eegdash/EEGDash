**Access the data**

```python
from eegdash.dataset import NM000204

dataset = NM000204(cache_dir="./data")
```

### Query & Filter

**Filter by subject**

```python
dataset = NM000204(cache_dir="./data", subject="01")
```

**Advanced query**

```python
dataset = NM000204(
    cache_dir="./data",

)
```

**Iterate recordings**

