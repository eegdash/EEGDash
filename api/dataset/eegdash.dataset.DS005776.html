DS005776#

Electrical_Thermal_FingerTapping_2015

Access recordings and metadata through EEGDash.

Citation: Ycel, Meryem, Selb, Juliette, Aasted, Christopher, Petkov, Mihayl, Borsook, David, Boas, David, Becerra, Lino (2025). Electrical_Thermal_FingerTapping_2015. 10.18112/openneuro.ds005776.v1.0.1

Modality: fnirs Subjects: 11 Recordings: 46 License: CC0 Source: openneuro

Metadata: Complete (90%)

Quickstart#

Install

pip install eegdash

Access the data

from eegdash.dataset import DS005776

dataset = DS005776(cache_dir="./data")
# Get the raw object of the first recording
raw = dataset.datasets[0].raw
print(raw.info)

Filter by subject

dataset = DS005776(cache_dir="./data", subject="01")

Advanced query

dataset = DS005776(
    cache_dir="./data",
    query={"subject": {"$in": ["01", "02"]}},
)

Iterate recordings

for rec in dataset:
    print(rec.subject, rec.raw.info['sfreq'])

Notes

Each item is a recording; recording-level metadata are available via dataset.description. query supports MongoDB-style filters on fields in ALLOWED_QUERY_FIELDS and is combined with the dataset filter. Dataset-specific caveats are not provided in the summary metadata.
