### Get Started

**Install**

```bash
pip install eegdash
```

**Access the data**

```python
from eegdash.dataset import NM000270

dataset = NM000270(cache_dir="./data")
# Get the raw object of the first recording

```

### Query & Filter

**Filter by subject**

```python
dataset = NM000270(cache_dir="./data", subject="01")
```

**Advanced query**

```python
dataset = NM000270(
```

**Iterate recordings**

