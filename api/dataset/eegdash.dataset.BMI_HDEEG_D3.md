### Get Started

**Install**

```bash
pip install eegdash
```

**Access the data**

```python
from eegdash.dataset import BMI_HDEEG_D3

dataset = BMI_HDEEG_D3(cache_dir="./data")
# Get the raw object of the first recording

```

### Query & Filter

**Filter by subject**

```python
dataset = BMI_HDEEG_D3(cache_dir="./data", subject="01")
```

**Advanced query**

```python
dataset = BMI_HDEEG_D3(

```

