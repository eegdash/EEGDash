## Quickstart

### Get Started

**Install**

```bash
pip install eegdash
```

**Access the data**

```python
from eegdash.dataset import Motion_Yucel2014

dataset = Motion_Yucel2014(cache_dir="./data")
# Get the raw object of the first recording
raw = dataset.datasets[0].raw
print(raw.info)
```

### Query & Filter

**Filter by subject**

```python
dataset = Motion_Yucel2014(cache_dir="./data", subject="01")
```

**Advanced query**

```python
dataset = Motion_Yucel2014(
    cache_dir="./data",
    query={"subject": {"$in": ["01", "02"]}},
)
```

**Iterate recordings**

```python

```
