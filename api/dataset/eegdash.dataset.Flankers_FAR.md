### Get Started

**Install**

```bash

```

**Access the data**

```python
from eegdash.dataset import Flankers_FAR

dataset = Flankers_FAR(cache_dir="./data")
# Get the raw object of the first recording

```

### Query & Filter

**Filter by subject**

```python
dataset = Flankers_FAR(cache_dir="./data", subject="01")
```

**Advanced query**

