### Get Started

**Install**

```bash

```

**Access the data**

```python
from eegdash.dataset import DS004817
```

### Query & Filter

**Filter by subject**

```python
dataset = DS004817(cache_dir="./data", subject="01")
```

**Advanced query**

```python
dataset = DS004817(
```
