Feature Engineering
===================

Three tutorials that teach EEGDash's feature extraction package as a
first-class option, not just deep learning. Sourced from
``docs/tutorial_restructure_plan.md`` Category E.

Run them in order:

1. ``plot_40_first_features.py`` — extract a small feature table from windows.
2. ``plot_41_feature_trees.py`` — use preprocessors and feature dependencies
   to avoid repeated computation.
3. ``plot_42_features_to_sklearn.py`` — train a scikit-learn / LightGBM
   baseline from feature tables.
