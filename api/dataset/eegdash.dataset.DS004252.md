## About This Dataset

This dataset contains electroencephalography (EEG) recordings from 16 participants viewing object images presented in eight different 2-D rotations. Stimuli were shown in rapid visual streams at two presentation rates (5 Hz and 20 Hz). EEG data are in standard bids format.
The pre-print can be found here: [https://doi.org/10.31234/osf.io/wp73u](https://doi.org/10.31234/osf.io/wp73u)
The analysis codes, results, and figures can be found on OSF: Moerel, D., Grootswagers, T., Robinson, A. K., Engeler, P., Holcombe, A. O., & Carlson, T. A. (2026, April 14). Rotation-tolerant representations elucidate the time course of high-level object processing. [https://doi.org/10.17605/OSF.IO/R93ES](https://doi.org/10.17605/OSF.IO/R93ES)
