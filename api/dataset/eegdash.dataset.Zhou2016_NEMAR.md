### Get Started

**Install**

```bash
pip install eegdash
```

**Access the data**

```python
from eegdash.dataset import Zhou2016_NEMAR

dataset = Zhou2016_NEMAR(cache_dir="./data")
# Get the raw object of the first recording
raw = dataset.datasets[0].raw
print(raw.info)
```

### Query & Filter

**Filter by subject**

```python
dataset = Zhou2016_NEMAR(cache_dir="./data", subject="01")
```

**Advanced query**


