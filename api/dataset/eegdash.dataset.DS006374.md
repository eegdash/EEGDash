**Install**

```bash

```

**Access the data**

```python
from eegdash.dataset import DS006374
```

### Query & Filter

**Filter by subject**

```python
dataset = DS006374(cache_dir="./data", subject="01")
```

**Advanced query**

```python
dataset = DS006374(
```
