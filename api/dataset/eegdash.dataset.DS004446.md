## Quickstart

### Get Started

**Install**

```bash
pip install eegdash
```

**Access the data**

```python
from eegdash.dataset import DS004446

dataset = DS004446(cache_dir="./data")
# Get the raw object of the first recording

```

### Query & Filter

**Filter by subject**

```python
dataset = DS004446(cache_dir="./data", subject="01")
```

**Advanced query**

```python
dataset = DS004446(
    cache_dir="./data",
```

**Iterate recordings**

